# Design Doc

## Vision

Key ideas:

- Fantasy-style RPG
  - Fight monsters
  - Do quests
- Open world
  - Can go anywhere in the world, monsters scale up with the player
- Top-Down 2D Pixel Graphics

Exclusions:

- Multiplayer
- 3D Graphics
- Mobile/Web
